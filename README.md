# c1visualizer
A mirror of the c1visualizer source code http://lafo.ssw.uni-linz.ac.at/c1visualizer/

ant build   
ant build-zip   

cd dist    
unzip c1visualizer-1.7.zip   
cd c1visualizer   
./bin/c1visualizer   

tips:   
delete /tmp/task.jar and re-run "ant build " when encountered error about task.jar
